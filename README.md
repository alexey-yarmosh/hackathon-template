### Commands:
```
Start dev:
meteor --settings settings-dev.json

Deploy:
meteor deploy baderfall.meteorapp.com --free --mongo --settings settings-prod.json

Save settings:
git-secret add <file>
git-secret remove <file>
git-secret hide
git-secret reveal
```
